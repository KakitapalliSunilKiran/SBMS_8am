What is Spring framework:
=========================
Spring is a open source Java based framework 
release in 2004 
current stable version is 6.x 
it is used to develop web applications
=========================================
Spring is developed by a company VM ware
Struts developed by a company Apache 
hibernate is developed by RedHat 
====================================

Spring has been developed in Modular fashion

Spring core 
Spring context 
Spring AOp 
Spring Jdbc 
Spring ORM 
Spring web 
Spring security 
Spring Batch 
Spring cloud 

Spring framework will not force all Modules to use 
There is No relationship between one Module to Other Module
